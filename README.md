# API & Front Installation 

### Back-End

Go to API directory 
    
    cd webApi

Install packages :

    npm install 

Launch : 

    npm start

ta taa !


### Front-End

Go to 
    
    cd webClient

Install packages : 
    
    npm install

Launch :
    
    ng serve -o 
    
You can access to the application with the following url : localhost:4200
